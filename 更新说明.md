# 欠料表处理工具 — 更新说明

> 版本对照：最初开发的 **v5.0**（Python + openpyxl 脚本版） → 当前 **v2.15**（Windows 桌面应用）

## 一、产品形态升级
- **从命令行脚本 → 图形界面应用**：v5.0 为 Python + openpyxl 脚本，需在命令行运行、手动传参；现重写为 C# / WPF 桌面程序，全程可视化操作。
- **单文件、免安装**：发布为单个 `欠料表处理工具.exe`，内置 .NET 运行时与所需原生组件，拷贝到任意 Windows 64 位电脑双击即可运行，无需安装环境。
- **专属应用图标**：白底圆角 + 粉紫渐变描边 + 表格对勾图标，资源管理器 / 任务栏 / 窗口标题栏统一显示。

## 二、界面与体验
- **全新白底粉紫渐变主题**：以白色为基底，主按钮粉色、次按钮紫色，头部粉→紫渐变，整体简约清爽。
- **布局优化**：每张卡片带统一加粗小标题（导出设置 / 回填设置），字段间距统一，主操作按钮左对齐；处理过程在底部「处理日志」中实时可见。
- **已移除**：暗色主题与右上角设置按钮（固定为白底粉紫，更聚焦）；独立的「导出 / 回填结果反馈」区（信息统一进入日志，界面更干净）。

## 三、导出功能
- 选择欠料表后**判定列自动加载**，无需手动找列。
- **类别手动选择**：外购 / 外协（按 W 列 Y/N 自动筛选；选「全部」则自动分目录）。
- **采购员筛选**（可选）：按采购员列筛选，留空则导出全部。
- 导出后**自动打开输出文件夹**，并保留「打开输出文件夹」按钮。

## 四、回填功能
- **默认目标列改为「采购交期回复」**（按列名自动定位），无需每次手动选列。
- **修复「回填后变数字」问题**：v5.0 时代，回复单里的 Excel 日期序列号（如 46084）常被原样写回而显示为数字；现自动识别日期序列号 / DateTime，转换为真正日期并套用日期格式，显示为日期；文本与普通数字（如供应商误填的 99999999）原样保留。

## 五、打包与部署
- 自包含单文件（约 71 MB），原生 `qiliao_core.dll` 已内嵌，磁盘上无散落文件。
- 如需更小体积（约 10 MB），可改用「框架依赖版」，但目标机需先安装 .NET 8 桌面运行时。
